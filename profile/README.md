## 🧠 Overview
**SiSoftware Sandra** is a comprehensive Windows utility that performs system analysis, hardware diagnostics, and benchmarking. It provides detailed insights into CPU, GPU, memory, storage, and network components to help users understand their system’s capabilities.  

Designed for both casual users and IT professionals, Sandra offers a range of performance tests, stress analyses, and system monitoring tools. Users can compare hardware, monitor real-time metrics, and optimize system performance effectively.
